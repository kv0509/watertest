# 黄玮恩喝水打卡应用

这是一个简单的喝水跟踪应用，帮助用户记录每日饮水量和保持健康的饮水习惯。

## 功能

- 记录每日饮水量
- 可视化水瓶显示当前进度
- 每小时饮水记录图表
- 每周饮水趋势图表
- 成就系统
- 数据本地存储

## 如何使用

1. 克隆或下载此仓库
2. 在浏览器中打开 `index.html` 文件
3. 开始记录您的饮水量！

## 部署到 GitHub Pages

1. 将整个仓库推送到 GitHub
2. 在仓库设置中启用 GitHub Pages
3. 选择 `main` 分支作为源
4. 访问生成的 URL 即可使用应用

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
- Chart.js (图表)
- Luxon (日期处理)
